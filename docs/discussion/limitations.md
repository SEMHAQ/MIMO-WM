# 局限性

## 我们承认的问题

- **还没在真机上验证**：MuJoCo再真实也是仿真，和真机有差距
- **合成数据集上精度低**：LSTM在简单场景下确实更好
- **R²只有0.592**：约41%方差未被解释，但这是任务本身的挑战（LSTM只有0.566）
- **推理时间比LSTM长**：在MuJoCo上9.5ms vs 5.0ms，FFT在高维状态下有开销
- **没做视觉输入**：目前只处理状态向量，不是端到端

## 失败场景

- 极短序列（T<16）：SSM无法充分利用长程依赖
- 剧烈碰撞：SSM的线性假设在不连续动力学下可能失效
- 分布外数据：训练和测试分布差异大时精度急剧下降

# 未来工作

## 六个方向

1. **视觉输入** — 结合CNN/ViT，从图像直接预测状态
2. **真机验证** — 在物理机器人上测试
3. **嵌入式部署** — NVIDIA Jetson上测量实际性能
4. **多模态融合** — 视觉+触觉+本体感觉
5. **结构化参数化** — 利用关节耦合拓扑设计SSM矩阵
6. **强化学习结合** — 端到端感知-决策-控制
