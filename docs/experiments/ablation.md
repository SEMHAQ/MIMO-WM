# 消融实验

## 核心问题：每个组件到底贡献了多少？

消融实验就是"拆零件"——把模型的一个组件去掉，看精度掉多少。

## 门控机制：贡献最大

| 去掉什么 | 合成数据集MSE变化 | MuJoCo MSE变化 |
|---------|----------------|---------------|
| 去掉门控 | +2.2% | +4.6% |
| 去掉残差连接 | +1.5% | +2.6% |

门控在MuJoCo上贡献更大——因为接触动力学有更多"不连续"，需要门控来选择性过滤信息。

## 网络深度和宽度

SSM-WM有三个关键超参数：

- **L（SSM层数）**：堆叠多少层SSM块，层数越多建模能力越强，但计算量也越大
- **D（隐空间维度）**：隐空间越大，模型能记住的信息越多，但参数量也越多
- **N（SSM状态维度）**：SSM内部状态的维度，影响每个SSM块的表达能力

| 配置 | MSE | 说明 |
|------|-----|------|
| L=2层 | 2.99 | 太浅，建模能力不足 |
| **L=4层（默认）** | **2.72** | 平衡点 |
| L=6层 | 2.64 | 再加层收益很小 |
| D=64 | 2.93 | 太窄 |
| **D=128（默认）** | **2.72** | 平衡点 |
| D=256 | 2.58 | 参数量增3.5倍，精度只改善5% |

**结论：L=4, N=16, D=128是最佳平衡点。**

## 门控阈值：软阈值最优

| 阈值类型 | MSE | 说明 |
|---------|-----|------|
| **软阈值（sigmoid）** | **0.834** | 连续输出，适应渐变 |
| Garrote | 0.852 | 稀疏但连续 |
| 硬阈值（0/1） | 0.891 | 二值信号，无法表达连续变化 |

硬阈值差6.4%——因为接触力是渐变的，不是"有或没有"。

## 训练损失：多步损失很关键

训练总损失 = (1-λ) × 单步损失 + λ × 多步损失

- **单步损失**：直接比较预测值和真实值的误差
- **多步损失**：用模型自己的预测继续往后展开H步，和真实轨迹比较
- **λ（多步损失权重）**：控制多步损失在总损失中的占比。λ=0表示完全不用多步损失，λ=1表示完全用多步损失

| λ（多步损失权重） | 单步MSE | 多步MSE (H=8) |
|---|---------|--------------|
| 0（仅单步损失） | 2.64 | 9.99 |
| **0.5（默认）** | **2.72** | **7.17** |
| 1.0 | 2.84 | 7.05 |

λ=0.5在单步精度和多步稳定性之间取得最佳平衡。多步损失让模型学会"自己的误差会怎么累积"，对MPC的多步展开预测至关重要。
